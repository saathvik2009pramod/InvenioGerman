# Digital Germany — Travel & Immigration Experience

Open `index.html` in your browser to explore the demo.

Features:
- Playful visa application form (fake)
- Learn basic German phrases with pronunciation (SpeechSynthesis)
- Short city guides: Berlin, Munich
- Tech in Germany overview

Usage:
1. Open the project folder in your file explorer.
2. Double-click `index.html` or serve it with a static server.

Notes:
- This is an educational demo only — not a real visa or immigration tool.
